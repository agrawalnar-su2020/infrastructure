## Infrastructure as Code with Terraform

### Install Terraform
- Download terrafrom from offical website HashiCrop
- Unzip it in root folder

### Install AWS command line interface 
- Install AWS cli
- Configuring AWS cli by setting up profile 

### Build Instructions
- Run commands 
- terraform initi
- terraform plan
- terraform apply

### Destroy Instruction 
- Run command
- terraform destroy
